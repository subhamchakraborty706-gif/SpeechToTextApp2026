Speech To Text Application

Description
A Speech-to-Text web application built using Java Spring Boot and HTML/JavaScript.

Features
- Speech recognition using microphone
- Speech-to-text conversion
- Spring Boot backend
- Simple web interface

Technologies Used
- Java
- Spring Boot
- HTML
- JavaScript

How to Run
1. Open the project in IntelliJ IDEA
2. Run DemoApplication
3. Open http://localhost:8080
